# 👑 Princess CyberGuide

**Cybersecurity Awareness Chatbot**  
A WPF desktop application built to educate users about online safety.

## ✨ Features

- Voice greeting on startup
- Modern chat interface with bubbles
- Smart keyword recognition using **Dictionary**
- Remembers user's name and interests (**List**)
- Multiple random responses per topic (**Arrays**)
- Good error handling for unknown inputs
- Many cybersecurity topics covered

## 🚀 How to Run

1. Open the solution (`PrincessCyberGuide.sln`) in **Visual Studio 2022**
2. Build and run the project (`F5`)
3. Place `CHATSOUND.wav` in the bin/Debug or bin/Release folder for voice greeting

## 📚 Topics the Bot Can Answer

- Password Safety
- Phishing
- Scams & Fraud
- Cyber Attacks / Ransomware
- Safe Browsing
- Online Privacy
- VPN, Antivirus, Firewall, Encryption, etc.

## 🛠️ Technologies Used

- C# .NET WPF
- Dictionary for keyword recognition
- List for storing user interests
- Arrays for multiple responses
- SoundPlayer for greeting
